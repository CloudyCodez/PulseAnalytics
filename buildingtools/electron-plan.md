# Pulse — Electron Desktop App Build Plan

## Goal
Wrap the existing Next.js Pulse app in Electron to produce a standalone
`Pulse.exe` that clients double-click to run. No terminal, no browser,
no Node knowledge required. The app opens in its own window and manages
everything internally.

Two build targets:
- `Pulse Demo.exe` — mock mode baked in, sample data, no keys needed
- `Pulse.exe` — real mode, configured per client during setup

## Run order to get Pulse.exe
1. Run `build-pulse.bat`
2. Press **3** → test Electron dev mode (Next.js must be running)
3. If window opens correctly → press **1** for Demo build
4. Installer lands in `D:\business\Pulse\dist\`
