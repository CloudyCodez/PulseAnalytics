Step 3 of 8 · Integration
Connect Google Analytics & Ads

Step 5 of 8 · Integration
Connect Shopify
Add your Shopify Admin API token so Pulse can pull orders, revenue, products, and customer data into your reports.
How to get your token: Shopify Admin → Settings → Apps → Develop apps → Create app → Configure Admin API → enable read_orders, read_products, read_customers, read_analytics → Install app → copy the Admin API access token.
Open Shopify guide →
Step 6 of 8 · Integration
Connect Klaviyo
Add your Klaviyo Private API key to include email marketing performance — open rates, click rates, flow revenue — in your Pulse reports.
How to get your key: Klaviyo → Account (bottom-left) → Settings → API Keys → Create Private API Key → set access to Read-only for Metrics, Campaigns, Flows, and Lists.
Open Klaviyo API docs →
